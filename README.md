Netnix - small react frontend for a collection of series
========================================================

**netnix**  
*noun* — invitation to sue me; wordplay on the name of a certain
streaming platform (misspelled Dutch for 'just nothing').

### Installation
Put this in the root directory of a web server with the nginx option
`autoindex_format json`.
